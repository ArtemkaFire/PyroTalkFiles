# PyroTalkFiles

This repository contains encrypted data for the **PyroTalk** messenger.

## 🔥 Download the App

The app is available for:
- 📱 Android (APK)
- 💻 Windows (EXE)
- 🐧 Linux (AppImage)
- 🍎 macOS (DMG)

All files can be downloaded from the Telegram channel:

👉 [My Telegram Channel](https://t.me/ArtemkaFireChannel) 👈

## 🔒 Privacy

All data is encrypted.

## 🌐 Language

The app interface is available in **Russian** and **English**.
Language can be changed in settings, and when you first log in to the app.and on the first visit to the app.
